================================================================================
推奨設定
================================================================================

デフォルト設定（推奨）:
  v_init=0.5    # 初期欠損（中央値）
  s_init=0.5    # 初期状態（中央値）
  eta=0.05      # 学習率（バランスの取れた値）
  v_min=0.1     # V の下限
  v_max=1.0     # V の上限

パラメータ調整ガイド:
  - eta=0.01: 非常に遅い収束 → 長期の細かい制御
  - eta=0.05: バランス型 → ほとんどの用途に推奨 ★
  - eta=0.1: 速い応答 → 迅速な状態変化が必要な場合
  - eta=0.2: 最も速い → 小規模システム向け（推奨外）
